Hypergeometric Distribution

Formula

\( P(X = x) = {{(_{x}^{M}) (_{n-x}^{N-M})} \over {(_{n}^{N})}} \)

where

P: probability

X: random variable

x: value of random variable

P(X = x): probability that random variable X has value x

M: # of successes in N

x: # of successes in n

N: population size

n: sample size
