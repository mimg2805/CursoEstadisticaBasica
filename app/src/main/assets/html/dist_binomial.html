Distribución Binomial

Fórmula

\( P(X = x) = {n! \over {x!(n - x)!}} π^x (1 - π)^{n-x} \)

donde

P: probabilidad

X: variable aleatoria

x: valor de variable aleatoria

P(X = x): probabilidad de que la variable aleatoria X tome un valor x

n: # de ensayos o tamaño de muestra

π: probabilidad de éxito
