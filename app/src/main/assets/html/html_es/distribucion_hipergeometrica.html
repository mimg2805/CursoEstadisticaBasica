Distribución Hipergeométrica

Fórmula

\( P(X = x) = {{(_{x}^{M}) (_{n-x}^{N-M})} \over {(_{n}^{N})}} \)

donde

P: probabilidad

X: variable aleatoria

x: valor de variable aleatoria

P(X = x): probabilidad de que la variable aleatoria X tome un valor x

M: # de éxitos en N

x: # de éxitos en n

N: tamaño de la población

n: tamaño de la muestra
