Selección Aleatoria de una Muestra

La mejor forma de seleccionar una muestra es aplicando el azar. De esta manera, la muestra seleccionada se realiza sin ninguna preferencia, teniendo todas las unidades de la población una probabilidad de ser seleccionada, y la misma para todos.

Las unidades a ser seleccionadas deben ser listadas y numeradas. A partir de esta lista y esta numeración, se pueden generar números aleatorios para saber qué unidades fueron seleccionadas.
